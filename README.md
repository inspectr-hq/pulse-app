# Pulse

Pulse is a lightweight macOS menu bar uptime checker, built as a subproject of [Inspectr](https://inspectr.dev/).

It gives you a fast, always-available way to monitor the health of endpoints you care about, right from your menu bar.

## Why Monitoring Matters

When you are building and shipping APIs, MCP servers, internal services, or public websites, small outages and slowdowns are easy to miss until users report them.

Basic uptime monitoring helps you:

- catch issues early
- verify deployments quickly
- track service responsiveness over time
- reduce debugging time when incidents happen

Even simple visibility can prevent long downtime windows.

## What Pulse Is

Pulse is intentionally small and focused:

- monitor multiple URLs from the macOS menu bar
- see up/down/checking status at a glance
- view response times and status codes
- run manual checks anytime
- review recent check history

Use it to watch:

- APIs
- MCP servers
- websites
- health endpoints
- staging environments

## Relationship to Inspectr

Pulse is a utility app in the Inspectr ecosystem. It complements broader tooling from [inspectr.dev](https://inspectr.dev/) by providing quick local monitoring during development and operations.

## Status

This is an active utility project and evolves iteratively based on real usage.
